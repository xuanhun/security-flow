# Report Template

Reports generated by `scripts/kali.py report` should include:

1. Assessment title and generation time.
2. Scope and exclusions.
3. Method summary.
4. Findings ordered by severity.
5. Evidence paths.
6. Recommendations.
7. Pending or unverified actions.
8. Appendix with tool status and command trace when appropriate.

Keep reports factual. Separate confirmed findings from hypotheses and pending
manual validation.
